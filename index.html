# 欢迎来到 pwn.hust.college `pwn.hust.college` 是一个基于宝可梦动画设计的趣味教育平台，供学生或其他感兴趣的团体学习和动手实践核心网络空间安全概念。同学们作为宝可梦训练家，通过挑战宝可梦道馆实践课堂中学到的网络空间核心安全攻防技术，获取相应道馆徽章，并最终成长为技术娴熟的白帽黑客。我们的设计理念为 **”未知攻，焉知防“** 与 **“实践出真知”**，旨在让同学们在道馆挑战中动手实践网络空间攻防的理论知识。 本平台由华中科技大学网络空间安全学院的出色[白帽黑客团队](#运维团队)维护。目前该平台支撑网安学院的软件安全理论课程与实践课程，且免费开源。 如果您有任何意见，建议和反馈，请通过[邮箱](mailto:pwn_hust_college@hust.edu.cn)联系我们!

# 宝可梦道馆 每个宝可梦道馆都有一个道馆馆主，同学们作为宝可梦训练家凭借自己的能力战胜道馆馆主的人将获得道馆徽章作为胜利的证明，只有集齐8枚徽章才能参加宝可梦联盟大会。宝可梦道馆使得青年宝可梦训练家通过挑战融会贯通学习到的理论知识。 在开展道馆挑战之前，各位宝可梦训练家需要完成如下所示教学关卡 - “pwn.hust.college 使用教学”，才能从真新镇大木博士研究所获得初始神奇宝贝 - 皮卡丘。开始学习吧，各位宝可梦训练家！

# 徽章（Badge） 徽章作为宝可梦训练家通过道馆主人考验的标志，是参加宝可梦联盟大会的入场券，同时，也是宝可梦训练家实力的象征。 为了激励大家学习热情，**我们将制作好真实的道馆徽章，并发放给完成道馆挑战的宝可梦训练家。**
## 如何获得灰色徽章？ 宝可梦训练家，你只有打败了深灰道馆馆主小刚的小拳石和大岩蛇才能完成道馆挑战，才能获得灰色徽章。 - [深灰道馆]()
## 如何获得蓝色徽章？ 宝可梦训练家，你只有打败了华蓝道馆馆主小霞的海星星和宝石海星才能完成道馆挑战，才能获得蓝色徽章。 - [华蓝道馆]()
## 如何获得橙色徽章？ 宝可梦训练家，你只有打败了枯叶道馆馆主马志士的雷丘才能完成道馆挑战，才能获得橙色徽章。 - [枯叶道馆]()
## 如何获得金色徽章？ 宝可梦训练家，你只有打败了金黄道馆馆主娜姿的凯西（进化为勇基拉）才能完成道馆挑战，才能获得金色徽章。 - [金黄道馆]()
## 如何获得彩虹徽章？ 宝可梦训练家，你只有打败了玉虹道馆馆主莉佳的蔓藤怪，口呆花与臭臭花才能完成道馆挑战，才能获得彩虹徽章。 - [玉虹道馆]()
## 如何获得粉红徽章？ 宝可梦训练家，你只有打败了浅红道馆馆主阿桔的毛球（进化为摩鲁蛾）与大嘴蝠才能完成道馆挑战，才能获得粉红徽章。 - [浅红道馆]()
## 如何获得深红徽章？ 宝可梦训练家，你只有打败了红莲道馆馆主夏伯的九尾，钻角犀兽与鸭嘴火兽才能完成道馆挑战，才能获得深红徽章。 - [红莲道馆]()
## 如何获得绿色徽章？ 宝可梦训练家，你只有打败了红莲道馆馆主坂木的怪力，巨钳蟹与钻角犀兽才能完成道馆挑战，才能获得深红徽章。 - [常青道馆]()
## 如何获得真实徽章？ 一旦你完成所有必需的挑战关卡，你可以通过 `pwn.hust.college` 平台注册账号的关联邮件账号发送邮件给我们，获取真实徽章！

# 合作与通关（Writeup）政策 `pwn.hust.college` 创建的挑战关卡是教育素材，用于给华科网安学院的学生评分。因此，我们希望不要在互联网上发布这些挑战关卡的通关攻略、讲解视频和直播。显然，我们无法阻止您在互联网上发布内容，但我们努力使所有这些内容都变成公开的，我们希望您的帮助能够使 `pwn.hust.college` 保持为一个可行的教育平台。

# 贡献 `pwn.hust.college` 基础架构及相关道馆仓库均开源在 [HUSTSeclab 组织中](https://github.com/HUSTSeclab/)，欢迎大家提交 PR 和 issue。然而，道馆仓库中可能不包含 write-up 或 solution。如果你是一位教育工作者，或者其他我们信任的人，并且对与模块本身素材的合作感兴趣，通过[邮箱](mailto:pwn_hust_college@hust.edu.cn)联系我们!。 如果您想参与贡献，请遵循”上游优先“原则，优先贡献由亚利桑那州立大学创建的上游 `pwn.collge` 平台。
