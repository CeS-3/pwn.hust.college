# 欢迎来到 pwn.hust.college `pwn.hust.college` 是一个基于宝可梦动画设计的趣味教育平台，供学生或其他感兴趣的团体学习和动手实践核心网络空间安全概念。同学们作为宝可梦训练家，通过挑战宝可梦道馆实践课堂中学到的网络空间核心安全攻防技术，获取相应道馆徽章，并最终成长为技术娴熟的白帽黑客。我们的设计理念为 **”未知攻，焉知防“** 与 **“实践出真知”**，旨在让同学们在道馆挑战中动手实践网络空间攻防的理论知识。 本平台由华中科技大学网络空间安全学院的出色[白帽黑客团队](#运维团队)维护。目前该平台支撑网安学院的软件安全理论课程与实践课程，且免费开源。 如果您有任何意见，建议和反馈，请通过[邮箱](mailto:pwn_hust_college@hust.edu.cn)联系我们!
# 宝可梦道馆 每个宝可梦道馆都有一个道馆馆主，同学们作为宝可梦训练家凭借自己的能力战胜道馆馆主的人将获得道馆徽章作为胜利的证明，只有集齐8枚徽章才能参加宝可梦联盟大会。宝可梦道馆使得青年宝可梦训练家通过挑战融会贯通学习到的理论知识。 在开展道馆挑战之前，各位宝可梦训练家需要完成如下所示教学关卡 - “pwn.hust.college 使用教学”，才能从真新镇大木博士研究所获得初始神奇宝贝 - 皮卡丘。开始学习吧，各位宝可梦训练家！

# 徽章（Badge） ## 如何获得真实徽章？ 一旦你完成所有必需的挑战关卡，你可以发送通过与 pwn.hust.college 账号关联的邮件账号发送邮件，来获取真实徽章！ # 合作与通关（Writeup）政策 `pwn.hust.college` 创建的挑战关卡是教育素材，用于给华科网安学院的学生评分。因此，我们希望不要在互联网上发布这些挑战关卡的通关攻略、讲解视频和直播。显然，我们无法阻止您在互联网上发布内容，但我们努力使所有这些内容都变成公开的，我们希望您的帮助能够使 `pwn.hust.college` 保持为一个可行的教育平台。
# 贡献 `pwn.hust.college` 基础架构及相关道馆仓库均开源在 [HUSTSeclab 组织中](https://github.com/HUSTSeclab/)，欢迎大家提交 PR 和 issue。然而，道馆仓库中可能不包含 write-up 或 solution。如果你是一位教育工作者，或者其他我们信任的人，并且对与模块本身素材的合作感兴趣，通过[邮箱](mailto:pwn_hust_college@hust.edu.cn)联系我们!。 如果您想参与贡献，请遵循”上游优先“原则，优先贡献由亚利桑那州立大学创建的上游 `pwn.collge` 平台。
